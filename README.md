# Scripts

## This repo contains api of Elaine Voice

### Install

`git clone <REPO>`
`cd <FOLDER>`

`python -m venv venv`
then activate venv

then install req's
`python -m pip install -r requirements.txt`

then start api
`uvicorn main:app`

or start it with watch/live-reload
`uvicorn main:app --reload`
